# 前端人工验收清单

这份清单用于验证自动化测试无法可靠判断的真实体验：触屏手感、语音实际播放、插图与文字的视觉关系、AI 内容语义，以及不同账号/浏览器的表现。

## 准备

- 准备两个普通账号：`账号 A`（生成者）和 `账号 B`（另一位读者）。
- 至少验证：iOS系统和Android系统分别验证。
- 使用默认系统字体和浏览器缩放；在最后一轮再开启较大系统字体。
- 每个问题记录：账号、设备/浏览器、主题、网络环境、操作步骤、截图或短录屏。

## 1. 基本阅读流程

- [x] 首页、登录、书架、阅读器、社区之间可进入和返回；浏览器返回键不会白屏、卡死或返回错误页面。
- [x] 浅色、纸张、夜间三种主题下，正文、按钮、标签和提示文案均清晰可读。
- [x] 调整字号、主题后，正文没有横向滚动；设置面板关闭后阅读位置不跳动。
- [x] 桌面与手机上，助手、证物袋可打开、切换、关闭；弹层不遮住关键正文，关闭后能继续阅读。

## 2. 证物与提示词

- [x] 点击一个正文线索后，证物袋数量、线索状态和对应证物卡同步更新。
- [x] 连续收集至少两个不同线索；第二个线索仍可点击，不会因第一个已收集而失效。
- [x] 在证物袋中确认：名称、类别、来源段落/文字与实际点击内容一致；删除或重置后状态同步恢复。
- [x] 打开证物图片/提示词生成入口，确认默认选中的证物是当前已收集且与场景相关的条目，不混入明显无意义的词、标点或不相关人名。
- [x] 更换证物、重新生成或恢复默认后，界面明确显示当前采用的证物和图片；阅读器、证物袋、对应正文状态保持一致。
- [x] 生成失败、网络慢或无可用证物时，有可理解的提示、重试/返回路径，按钮不会永久加载或失效。

## 3. 插图版本与账号隔离

- [x] 用新账号 B 首次进入同一章节：只显示官方默认插图，不出现其他用户刚生成的私有图片。
- [x] 账号 A 在某一段生成插图后，A 的阅读器显示新图；刷新页面后仍保持 A 自己的当前选择。
- [x] 在账号 B 中打开相同段落：仍显示默认图，不应自动同步 A 的新图。
- [x] A 主动采用公开插图后，仅 A 的阅读器切换为该版本；B 仍不受影响，除非 B 自己也采用。
- [x] 非图片作者不出现“删除/移出我的版本”等所有权操作；作者自己的版本可正常撤回、删除或恢复官方版本。
- [x] 恢复官方插图后，图片回到默认版本，段落位置正确，刷新后不会重新出现已移除的个人图。
- [x] 图片紧跟其对应段落，不插入前后错误段落；收起/展开不造成文字跳行、重叠或大面积空白。

## 4. 语音与其他多模态互动

- [x] 每个有语音的段落/对白只显示一个播放入口；没有重复的网页播放按钮或错位图标。
- [x] 点击播放、暂停、切换到另一段语音时，前一段会停止，页面不会出现两个声音同时播放。
- [x] 语音加载中、加载失败、无音频时有明确反馈；不会因为失败导致正文或其他按钮无法操作。
- [x] 语音按钮在手机上易点，不遮住段落首尾；滚动、切换主题或调整字号后位置仍正确。
- [x] 图片、语音、证物袋、助手连续使用后，阅读器仍流畅，正文位置不无故跳回顶部。

## 5. 选段、阅读手感与可访问性

- [x] 长按正文约半秒后，可以稳定选中当前完整段落。
- [x] 选中后，高亮范围完整，固定边界标记与文字边缘对齐；工具条不覆盖主要文字，按钮可一次点中。
- [x] 选中后滚动、切换横竖屏，选区与工具条不会明显错位。
- [x] 键盘操作桌面端：Tab 焦点清晰可见；标签可用方向键/Home/End 切换；Escape 能关闭弹层并回到触发按钮。

## 6. 移动端与视觉体验

- [x] 竖屏、横屏各检查一次；顶部导航、底部安全区、浏览器工具栏不遮挡正文与阅读辅助操作。
- [x] 开启较大系统字体后，标题、正文、按钮、图片说明和弹层不重叠、不被裁切。
- [x] Android Chrome、Android 夸克、iPhone 浏览器中分别检查：首页加载、登录、阅读器打开、图片显示、语音按钮显示。
- [x] 页面首屏和打开阅读器后无长时间空白；慢网络下有加载反馈。

## 7. 导航、账号、书架与个人资料

- [x] 顶部品牌、首页、创作广场、书架、个人资料、登录/注册入口均能到达正确页面；当前页面状态清晰。
- [x] 从书架、个人资料、创作广场和阅读器返回时，统一返回按钮回到合理的上一层，不出现死路或意外回到登录页。
- [x] 登录错误、注册字段缺失、重复提交、退出登录均有清晰反馈；退出后不再看到该账号的个人封面、证物和创作版本。
- [x] 书架中的阅读进度、最近阅读章节、证物数量与阅读器实际状态一致；继续阅读会打开正确章节。
- [x] 个人资料中的当前封面、已收集证物、个人创作入口和“去设计封面”跳转正确，不显示别人的私有内容。

## 8. 封面工作台与封面社区

- [x] 阅读器的“封面设计”可在快捷生成、自由创作、我的版本、封面社区之间稳定切换；切换不会丢失已填写的内容。
- [x] 快捷生成中的人物、阵容、关系、戏剧节点、景别、机位、光源、色彩、质感等控制项能选中，并且“画面设定”摘要随之更新。
- [x] 当文字描述与镜头设定冲突时，界面明确说明约束；封面文字（书名、作者、副标题）可编辑且在预览中不被裁切。
- [x] 生成中有明确等待状态，不能重复提交；成功后出现版本号、预览、提示词、采用/发布操作；失败后仍可返回编辑并保留输入。
- [x] “采用为当前封面”后，首页、书架和阅读器显示同一封面；“恢复官方封面”后，三处同步恢复，历史版本仍保留。
- [x] 封面社区的热门/最新排序、点赞、收藏、撤回、举报、查看提示词、基于此作品继续创作均有清晰结果；非作者不出现撤回操作。
- [x] 手机上封面编辑表单、标签组、文本框、版本缩略图与社区卡片可滚动操作，不发生横向溢出或误触。

## 9. 段落创作面板、插图与配音

- [x] 选中段落后，工具条中的“生成插图、我来创作插图、创作广场、一键生成 AI 配音、我来配音、我来设计音色、我来设计 AI 配音”均可打开对应面板或给出明确的暂不可用说明。
- [x] 打开一个面板再打开另一个时，上一个面板会收起；关闭按钮、遮罩点击和手机底部抽屉关闭都能恢复正文阅读。
- [x] 插图创作支持官方风格/自由创作、查看我的版本、生成、发布/撤回、采用、恢复官方；切换版本不会误改其他段落。
- [x] AI 配音设计中，角色、音色、台词片段、参数与生成状态易理解；连续打开/关闭面板不会残留播放控件或遮挡正文。
- [x] 真人配音入口、音色设计、已采用配音、公共配音的状态明确；权限不足、麦克风拒绝、浏览器不支持时有可理解反馈。
- [x] 选段的插图与配音在同一段落内排列有序：不盖住评论、证物线索或下一段文字；滚动后仍与原段落关联。

## 10. 创作广场与社区互动

- [x] 创作广场的配音社区、插图社区、证物社区、封面社区四个分区均可切换；首次加载、空状态、加载失败和重试状态可理解。
- [x] 各社区的“热门/最新/全部/我的作品”等筛选与排序生效，切换后不会保留前一分区的错误列表或加载状态。
- [x] 每种作品卡片的作者、来源段落、类型、状态、提示词、点赞数/采用数等信息可读；长提示词、长用户名和较窄手机屏幕不撑破卡片。
- [x] 非作者可点赞、采用、收藏或举报（按页面提供的功能）；作者可撤回自己的作品；重复点击不会让计数跳动、按钮卡在加载中或产生重复作品。
- [x] 从社区的“查看原文/回到阅读”进入阅读器时，应定位到正确章节和对应段落；从阅读器返回社区时，分区与筛选状态尽量保留。
- [x] 公开、撤回、采用、取消采用后，刷新页面验证状态真实保存；账号 A 与 B 的“我的作品/当前采用”不能串号。

## 11. 助手、评论与反馈边界

- [x] 案情助手可发送问题、按 发送 提交、显示回复和加载/失败状态；空问题不会发送；回答不应遮挡输入框或导致阅读器跳页。
- [x] 段落评论可展开、收起、发布、刷新后保留；自己的评论可删除，其他人的评论不显示删除按钮。
- [x] 评论的未登录提示会正确跳转登录；取消登录或返回后不会丢失阅读位置。
- [x] 举报时取消输入不会提交；输入过短、过长、网络失败或重复提交有明确提示，不应改变原作品可见状态。
- [x] 所有 AI 生成、发布、采用、删除、举报操作在成功、失败、等待中都有可见反馈；不应只依靠按钮颜色判断状态。

## 12. 移动端专属阅读控制

- [x] 手机阅读器顶部返回、目录/章节入口、阅读设置和底部操作栏在单手范围内可点；展开阅读辅助后能收回，不长期遮挡正文。
- [x] 手机目录抽屉可打开、选择章节、关闭；当前章节与进度标记正确，切章后正文回到合理位置。
- [x] 手机上的段落创作、社区选择与阅读辅助采用底部抽屉时，安全区、键盘弹出和浏览器底栏不遮住输入框、关闭按钮或主操作。
- [x] 横竖屏切换、返回键、浏览器前进/后退后，抽屉和遮罩不会残留，页面仍可滚动与点击。

## 通过标准与记录

- 核心流程、账号隔离、证物/插图位置、播放入口重复问题必须全部通过，才可认为本轮前端验收通过。
- AI 提示词的“是否有意义”按人工语义判断记录；遇到边界情况应附上原文、所选证物和生成结果，不以单次失败直接判定系统故障。
- 动态问题（播放重叠、选区闪烁、滚动错位）优先录屏；静态问题使用截图。
