# 沉浸式多模态阅读器

一个面向探案类小说的沉浸式阅读网站原型。当前研究目标以《福尔摩斯探案集》为例，先实现“可阅读、可对话、有画面、有语音入口、有背包悬浮球”的 Web MVP。

## 当前 MVP 功能

- 阅读器页面：章节选择、阅读进度、正文阅读。
- 场景画面：根据当前段落展示案件场景卡片。
- 语音入口：为正文和角色对话预留朗读/配音接口。
- 对话助手：读者可以围绕当前案情提问，先使用模拟回复。
- 悬浮背包：收集线索、人物、地点等探案信息。
- 后端 API：提供章节、场景、背包、对话等接口骨架。

## 技术栈

- 前端：React + TypeScript + Vite
- 后端：Node.js + Express
- 当前数据：本地 mock 数据，后续可替换为数据库和真实 AI 服务

## 目录结构

```text
frontend/   # 阅读网站前端
backend/    # API 服务
docs/       # 项目文档
assets/     # 素材与演示资源
```

## 本地运行

先安装 Node.js，然后在项目根目录执行：

```bash
npm install
npm run dev
```

运行后：

- 前端：http://localhost:5173
- 后端：http://localhost:3001/api/health
