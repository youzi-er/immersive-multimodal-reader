# 沉浸式多模态阅读器

将传统、文字密集型网络小说转化为可探索的沉浸式阅读体验。读者不再只是旁观者，而是在故事场景中行动的角色：系统根据当前情节在关键切换点生成图片、环境音与交互反馈，并用 AI 串联连续的叙事体验。

## 项目目标

- 在阅读场景切换时提供与剧情一致的 AI 视觉素材与音频氛围。
- 让读者以第一人称参与剧情，完成轻量交互与分支选择。
- 将排版、文字、图片、声音和交互整合为流畅的多模态阅读界面。
- 通过用户测试评估可用性、沉浸感与叙事连贯性。

## MVP 范围

1. 小说章节导入与阅读器基础界面。
2. 依据章节/场景元数据展示图片、背景音与交互卡片。
3. AI 服务接口层：支持生成或接入场景图片、音频与文本摘要。
4. 用户阅读进度、互动选择和基础反馈记录。
