# 腾讯云部署记录

这份文档用于后续把网站部署到腾讯云服务器。当前先完成本地开发，等本地功能稳定后再部署。

## 服务器需要的信息

部署时需要你提供：

- 服务器公网 IP
- 登录用户名，通常是 `root` 或 `ubuntu`
- 登录方式：密码，或 SSH 私钥文件
- 服务器系统：建议 Ubuntu 22.04 / 24.04
- 域名：如果暂时没有，可以先用 IP 访问

不要直接把密码写进 GitHub 仓库。

## 目标部署方式

第一版建议：

- 前端：Vite 构建后生成静态文件，由 Nginx 提供访问
- 后端：Node.js Express 服务，由 PM2 常驻运行
- 反向代理：Nginx 将 `/api` 转发到后端

## 基本步骤

1. 登录腾讯云服务器。
2. 安装 Node.js、Git、Nginx、PM2。
3. 拉取 GitHub 仓库代码。
4. 执行 `npm install` 和 `npm run build`。
5. 配置 Nginx。
6. 使用 PM2 启动后端。
7. 打开腾讯云安全组端口：80、443、3001。

具体命令等确认服务器系统和登录方式后再写。
